Котлеты

На сковородку одновременно можно положить k котлет. Каждую котлету нужно с каждой стороны обжаривать m минут непрерывно. За какое наименьшее время удастся поджарить с обеих сторон n котлет?

Входные данные
Вводятся 3 числа: k, m и n. Все числа не превосходят 32000.

Выходные данные
Вывести время, за которое все котлеты будут обжарены.

Sample Input:
1
5
1

Sample Output:
10

Напишите программу. Тестируется через stdin → stdout

# java
import java.lang.*;
import java.util.*;

class Meatballs {

  public static void main(String[] args) {

    Scanner scan = new Scanner(System.in);

    int panCapacity = scan.nextInt();
    int cookingTime = scan.nextInt();
    int meatballsAmount = scan.nextInt();

    scan.close();

    if (meatballsAmount == 0) {
      System.out.println(0);
    } else if (meatballsAmount <= panCapacity) {
      System.out.println(cookingTime * 2);
    } else {
      System.out.println((int) Math.ceil((double) meatballsAmount * 2 / panCapacity) * cookingTime);
    }

  }

}
