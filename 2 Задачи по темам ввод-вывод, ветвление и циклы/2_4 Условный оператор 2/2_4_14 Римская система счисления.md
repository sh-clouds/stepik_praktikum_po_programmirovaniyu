Римская система счисления

Дано число X. Требуется перевести это число в римскую систему счисления.

Входные данные
Дано число X в десятичной системе счисления (1  ≤  X  ≤  100).

Выходные данные
Выведите X в римской системе счисления.

Sample Input:
4

Sample Output:
IV

Напишите программу. Тестируется через stdin → stdout

$arab =  [1,   4,   5,   9, 10,  40, 50,  90, 100, 400, 500, 900, 1000]
$roman = ['I','IV','V','IX','X','XL','L','XC','C','CD','D','CM','M']

def arabToRoman(number)
    ret = ''
    i = $arab.length - 1 
    while(number > 0) 
        if(number >= $arab[i])
            ret += $roman[i]
            number -= $arab[i]
        else
             i-=1
        end
    end
    return ret
end

    public static string ToRoman(int number) {

   string result = "";
   int N = number; 
   int i = 8; 
   while (N > 0) 
   { 
   while (table[i].number > N) 
   { i--; }  
   result += table[i].symbol; 
   N -= table[i].number;
