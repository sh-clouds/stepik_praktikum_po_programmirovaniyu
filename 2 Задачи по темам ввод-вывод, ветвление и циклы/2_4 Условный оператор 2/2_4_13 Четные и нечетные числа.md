Четные и нечетные числа

Даны три целых числа A, B, C. Определить, есть ли среди них хотя бы одно четное и хотя бы одно нечетное.

Входные данные
Числа A, B, C, не превышающие по модулю 10000.

Выходные данные
Одна строка – "YES" или "NO".

Sample Input:
3
4
5

Sample Output:
YES

Напишите программу. Тестируется через stdin → stdout

import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int a = in.nextInt(), b = in.nextInt(), c = in.nextInt();
        in.close();
        boolean parityCheck_a = a % 2 == 0;
        boolean parityCheck_b = b % 2 == 0;
        boolean parityCheck_c = c % 2 == 0;
        if ((parityCheck_a && parityCheck_b && parityCheck_c) || (!parityCheck_a & !parityCheck_b && !parityCheck_c)) {
            System.out.println("NO");
        } else System.out.println("YES");

    }
}
