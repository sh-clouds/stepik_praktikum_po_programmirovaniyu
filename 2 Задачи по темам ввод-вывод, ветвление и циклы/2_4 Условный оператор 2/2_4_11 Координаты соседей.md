Координаты соседей

Для клетки с координатами (x, y) в таблице размером M × N выведите координаты ее соседей. Соседними называются клетки, имеющие общую сторону.

Входные данные
Даны натуральные числа M, N, x, y (1 ≤ x ≤ M ≤ 109, 1 ≤ y ≤ N ≤ 109).

Выходные данные
Выведите пары координат соседей этой клетки в порядке слева направо сверху вниз.

Sample Input:
3 3
2 2

Sample Output:
1 2
2 3
2 1
3 2

# C
#include <stdio.h>

int rangeOk(int x, int a, int b) {
    return a <= x && x <= b;
}

int main() {
    int m, n, x, y;
    scanf("%d %d %d %d", &m, &n, &x, &y);
    
    int P[][2] = {{x - 1, y}, {x, y + 1}, 
                  {x, y - 1}, {x + 1, y}};
    
    for (int i = 0; i < sizeof(P) / sizeof(*P); i++) {
        x = P[i][0], y = P[i][1];
        if (rangeOk(x, 1, m) && rangeOk(y, 1, n))
            printf("%d %d\n", x, y);
    }
    return 0;
}
