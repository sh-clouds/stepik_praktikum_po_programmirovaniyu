Послезавтра

По заданной дате требуется определить, какое число будет послезавтра.
Напомним, что год является високосным, если его номер кратен 4, но не кратен 100, а также если он кратен 400.

Входные данные
Дано число, месяц и год (год  – число в промежутке от 1 до 10000).

Выходные данные
Требуется вывести, какое число будет послезавтра, в формате входных данных.

Sample Input:
1 8 2009

Sample Output:
3 8 2009

Напишите программу. Тестируется через stdin → stdout

# C
#include <inttypes.h>
#include <stdio.h>

int main() {
    int8_t months[12] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    int64_t dd, mm, yyyy;
    scanf("%"SCNd64" %"SCNd64" %"SCNd64, &dd, &mm, &yyyy);

    months[1] += ((yyyy % 4 == 0 && yyyy % 100 != 0) || yyyy % 400 == 0);
    dd += 2;
    if (dd > months[mm - 1]) {
        dd -= months[mm - 1];
        mm++;
    }
    if (mm == 13) {
        yyyy++;
        mm = 1;
    }

    printf("%"PRId64" %"PRId64" %"PRId64, dd, mm, yyyy);
    return 0;
}
