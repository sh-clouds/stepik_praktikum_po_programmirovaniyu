Проверка на простоту

Проверьте, является ли число простым.

Входные данные
Вводится одно натуральное число n не превышающее 2000000000 и не равное 1.

Выходные данные
Необходимо вывести  строку prime, если число простое, или composite, если число составное.

Sample Input:
5

Sample Output:
prime

Напишите программу. Тестируется через stdin → stdout
Верно решили 128 учащихся
Из всех попыток 52% верных

# ruby
def is_prime?(n)
    if n < 4 then return n > 1 end
    k = 2
    while k*k <= n do
        if n%k == 0 then return false end 
        k += 1
    end        
    return true  
end        

n = gets.to_i
print is_prime?(n)==true ? "prime" : "composite"

let isprime = (w) => {
    if (w == 2) 
        return 'prime'
    else if (w % 2 == 0)
        return 'composite';
    
    for (let i = 3; i <= Math.sqrt(w); i += 2)
        if (w % i == 0)

    return 'prime';

#include <iostream>

bool isprime(unsigned long long n) {
    unsigned long long k;
    if (n<4) return true;
    if (n%2==0) return false;
    k=sqrt(n);
    for(unsigned long long i=3;i<=k;i+=2) if (n%i==0) return false;
    return true;
}
