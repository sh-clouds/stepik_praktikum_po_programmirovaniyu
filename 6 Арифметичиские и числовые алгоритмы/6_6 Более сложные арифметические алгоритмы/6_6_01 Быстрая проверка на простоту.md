Входные данные
Дано натуральное число N, 2 ≤ N ≤ 1018.

Выходные данные
Выведите YES, если число N является простым или NO, если число N является составным.

Sample Input 1:
2

Sample Output 1:
YES

Sample Input 2:
3

Sample Output 2:
YES

# ruby
def is_prime?(n)
    if n < 4 then return n > 1 end
    k = 2
    while k*k <= n do
        if n%k == 0 then return false end 
        k += 1
    end        
    return true  
end

let isprime = (w) => {
    if (w == 2) 
        return 'YES'

        return 'NO';
    
    for (let i = 3; i <= Math.sqrt(w); i += 2)

    return 'YES';
