N^N mod 10^P

Входные данные
Даны два числа N и P (1≤N≤109, 1≤P≤100).

Выходные данные
Вычислите N**N mod 10**P.

Sample Input:
5 3

Sample Output:
125

Напишите программу. Тестируется через stdin → stdout
Верно решили 54 учащихся
Из всех попыток 18% верных

# ruby
modulus = 10**b
print(a.pow(a, modulus))



# python
a, b = map(int, input().split())
modulus = 10**b
result = pow(a, a, modulus)
print(result)
