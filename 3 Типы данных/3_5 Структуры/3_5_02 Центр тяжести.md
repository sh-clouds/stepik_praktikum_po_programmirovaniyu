Центр тяжести

Выведите координаты центра тяжести данного множества точек.

Входные данные
Программа получает на вход набор точек на плоскости. Сначала задано количество точек n, затем идет последовательность из n строк, каждая из которых содержит два числа: координаты точки. Величина n не превосходит 100, все исходные координаты – целые числа, не превосходящие 10^3.

Выходные данные
Выведите  координаты центра тяжести данного множества точек. Ответ - пара действительных чисел с пробелом между ними, необходимо выводить с точностью до 10 значащих цифр после запятой. При этом ответ не надо округлять, лишние цифры просто отбрасываются.

Sample Input:
2
1 2
2 3

Sample Output:
1.5 2.5

Напишите программу. Тестируется через stdin → stdout

n = int(input())

def crutch(num: float) -> float:
    """Костыль, чтобы отбросить незначущую часть без округления"""
    return int(num * 1e10) / 1e10

total_x, total_y = 0, 0
for _ in range(n):
    x, y = map(float, input().split())
    total_x += x
    total_y += y
print(crutch(total_x / n), crutch(total_y / n))

import java.math.BigDecimal;
import java.util.ArrayList;
import java.util.List;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        int number = Integer.parseInt(input.nextLine());
        List<Point> points = new ArrayList<>();

        for (int a = 0; a < number; a++) {
            points.add(new Point(input.nextInt(), input.nextInt()));
        }

        Point.getMassOfPoints(points);
    }
}

class Point {
    private int x;
    private int y;

    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }

    public static void getMassOfPoints(List<Point> points) {
        double aX = 0d, aY = 0d, m = points.size();

        for(Point p : points) {
            aX += p.x;

        aX = aX / m;
        aY = aY / m;

        if(Double.toString(aX).substring(Double.toString(aX).indexOf(".") + 1).length() > 10) {
            System.out.print(new BigDecimal(aX).setScale(10, BigDecimal.ROUND_DOWN) + " ");

            System.out.print(aX + " ");

    }
