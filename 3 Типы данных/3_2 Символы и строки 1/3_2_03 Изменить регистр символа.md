Изменить регистр символа

Измените регистр символа, если он был латинской буквой: сделайте его заглавным, если он был строчной буквой и наоборот. 

Любой другой символ выведите как есть, ничего не меняя.

Входные данные
Задан единственный символ C.

Выходные данные
Необходимо вывести  получившийся символ.

Sample Input 1:
V

Sample Output 1:
v

Sample Input 2:
a

Sample Output 2:
A

Напишите программу. Тестируется через stdin → stdout

# JS
var stdin = process.openStdin();

stdin.addListener("data", function(d) {
    let result = d.toString();
    let code = result.charCodeAt();

    if ( code >= 65 && code <= 90 ) result = result.toLowerCase();
    if ( code >= 97 && code <= 122 ) result = result.toUpperCase();

    console.log(result);
  });

# java
import java.util.Scanner;
class Main {
    public static void main(String[] args) {
        char ch = new Scanner(System.in).next().charAt(0);
        System.out.print(ch >= 'a' && ch <= 'z' ? Character.toUpperCase(ch) :
                         ch >= 'A' && ch <= 'Z' ? Character.toLowerCase(ch) : ch);
    }
}


# C
#include <ctype.h>
#include <locale.h>
#include <stdio.h>
#include <stdlib.h>
#include <wchar.h>

int main() {
  setlocale(0, "");
  wchar_t c;
  if (scanf("%lc", &c) != 1) return EXIT_FAILURE;
  if (islower(c))
    printf("%lc\n", toupper(c));
  else if (isupper(c))
    printf("%lc\n", tolower(c));
  else
    printf("%lc\n", c);
  return EXIT_SUCCESS;
}
