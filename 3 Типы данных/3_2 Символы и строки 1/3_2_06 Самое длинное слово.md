Самое длинное слово

Дана строка, содержащая пробелы. Найдите в ней самое длинное слово, выведите  это слово и его длину. Если таких слов несколько, выведите первое из них.

Входные данные
Задана одна строка, содержащая пробелы. Слова разделены ровно одним пробелом. Пробелы в начале и конце строки допускаются.

Выходные данные
Необходимо вывести самое длинное слово в строке и его длину.

Sample Input:
one two three four five six
Sample Output:

three
5

Напишите программу. Тестируется через stdin → stdout

#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#define BUFF_SIZE 128

int main() {
  size_t res = 0;
  char str[BUFF_SIZE];
  char word[BUFF_SIZE];
  fgets(str, BUFF_SIZE, stdin);
  size_t len = strlen(str);
  for (size_t i = 0; i < len;) {
    size_t wordlen = strcspn(str + i, " ");

    if (wordlen > res) {
      res = wordlen;
      strncpy(word, str + i, wordlen);
      word[wordlen] = '\0';
    }
    i = i + wordlen + 1;
  }
  printf("%s\n%zu\n", word, res);
  return EXIT_SUCCESS;
}


# C++
#include <iostream>

int main() {
    std::string s, word = "", max_word = "";
    std::getline(std::cin, s);
    for (auto el : s) {
        if (el == ' ') {
            word = "";
            continue;
        }
        word += el;
        if (word.size() > max_word.size()) max_word = word;
    }
    std::cout << max_word << "\n" << max_word.size() << "\n";
    return 0;
}
