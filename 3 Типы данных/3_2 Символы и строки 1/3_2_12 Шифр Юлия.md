Шифр Юлия

Юлий Цезарь использовал свой способ шифрования текста. Каждая буква заменялась на следующую по алфавиту через K позиций по кругу. Необходимо по заданной шифровке определить исходный текст.

Входные данные
В первой строке дана шифровка, состоящая из заглавных латинских букв. Во второй строке число K (1 ≤ K ≤ 10).

Выходные данные
Требуется вывести результат расшифровки.

Sample Input 1:
XPSE
1

Sample Output 1:
WORD

Sample Input 2:
ZABC
3

Sample Output 2:
WXYZ
Напишите программу. Тестируется через stdin → stdout

# python
def f(text: str, n: int) -> str:
    res = ''
    for letter in text:
        res += chr((ord(letter) - 97 - n % 26) % 26 + 97)
    return res

print(f(input().lower(), int(input())).upper())

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String str = sc.next();
        int k = sc.nextInt();
        for (int i = 0; i < str.length(); i++) {
            int x =  str.charAt(i) - k;
            x=x<'A'?x+26:x;

    }

# C
#include <stdio.h>

int main(int argc, char **argv)
{
    char s[20] = {0};
    int n, l;
    char r;

    scanf("%s%n", s, &l);
    scanf("%d", &n);

    for (int i = 0; i < l; i++) {
        r = (s[i] - n) % 90;
        putchar(r < 65 ? r + 26 : r);
    }

    putchar('\n');

    return 0;
}
