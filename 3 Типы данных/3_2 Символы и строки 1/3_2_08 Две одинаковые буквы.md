Две одинаковые буквы

Дана строка. Известно, что она содержит ровно две одинаковые буквы. Найдите эти буквы. Гарантируется, что повторяются буквы только одного вида.

Входные данные
На вход подается 1 строка.

Выходные данные
Необходимо вывести  букву, которая встречается в строке дважды.

Sample Input:
fif

Sample Output:
f

Напишите программу. Тестируется через stdin → stdout

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String s = scanner.nextLine();
        int i = 0;
        int j = s.length() - 1;
        while (i != j) {
            String tmp = String.valueOf(s.charAt(i));
            if (s.indexOf(tmp) != s.lastIndexOf(tmp)) {
                System.out.println(tmp);
                break;
            }
            i++;
        }

# C
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#define BUFF_SIZE 128

int main() {
  char res = '\0';
  char str[BUFF_SIZE];
  scanf("%s", str);
  size_t len = strlen(str);
  for (int i = 0; i < len - 1; ++i) {
    for (int j = i + 1; j < len; ++j) {
      if (str[i] == str[j]) {
        res = str[i];
        printf("%c\n", res);
        return EXIT_SUCCESS;
      }
    }
  }
  return EXIT_FAILURE;
}
