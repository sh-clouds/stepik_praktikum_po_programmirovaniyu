Совпадают ли строки?

Необходимо определить, совпадают ли 2 строки.

Входные данные
Заданы 2 строки.

Выходные данные
Необходимо вывести  слово yes, если строки совпадают, и слово no в противном случае.

Sample Input:
a
a

Sample Output:
yes

Напишите программу. Тестируется через stdin → stdout

class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

    }
}


# C
#include <stdio.h>

int main(void) {
    char str1[127], str2[127];
    fgets(str1, 127, stdin);
    fgets(str2, 127, stdin);
    int res = 1;
    for (int i = 0; i < 127; i++) {
        if ((!str1[i] || str1[i] == '\n') && (!str2[i] || str2[i] == '\n')) break;
        if (str1[i] != str2[i]) {
            res = 0;
            break;
        }
    }
    printf("%s", res ? "yes" : "no");
    return 0;
}
